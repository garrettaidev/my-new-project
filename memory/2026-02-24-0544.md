## How it works

The skill uses the `youtube-transcript-api` Python library to:
1. Extract video IDs from various YouTube URL formats
2. Fetch available transcripts (including auto-generated captions)
3. Format them as markdown with optional timestamps
4. Save to a file with metadata header

## Quick start

### 1. Install dependencies
```bash
pip install youtube-transcript-api
```

### 2. Basic usage
```bash
python skills/public/youtube-transcript/scripts/fetch_transcript.py "https://www.youtube.com/watch?v=VIDEO_ID"
```
Creates `VIDEO_ID.md` in current directory.

### 3. Save to your GitHub repo
```bash
cd /path/to/your/repo
python /path/to/skill/scripts/fetch_transcript.py "YOUTUBE_URL" docs/transcript.md
git add docs/transcript.md
git commit -m "Add YouTube transcript"
git push
```
